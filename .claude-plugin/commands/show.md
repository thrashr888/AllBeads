Show detailed information about a specific bead.

Run `allbeads show <bead-id>` to display:
- Full title and description
- Status and priority
- Context (repository)
- Dependencies (blocks/blocked-by)
- Creation and update timestamps
- Comments (if any)

This provides complete context for understanding a piece of work, including cross-repo relationships.
