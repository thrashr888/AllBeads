---
description: List beads across all contexts
---

List beads aggregated from all configured AllBeads contexts.

Run `allbeads list` to show all beads. Supports filtering:
- `allbeads list --status open` - Filter by status
- `allbeads list --priority P1` - Filter by priority
- `allbeads list --context myproject` - Filter by context

Present results grouped by context when showing multiple repositories.
