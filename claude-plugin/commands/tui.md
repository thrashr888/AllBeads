---
description: Launch interactive TUI dashboard
---

Launch the AllBeads terminal user interface for interactive navigation.

Run `allbeads tui` to open the dashboard with:
- **Kanban view**: Visual board with Open, In Progress, and Closed columns
- **Mail view**: Agent messages and notifications

Keyboard shortcuts:
- `Tab` - Switch between Kanban and Mail views
- `j/k` or `Up/Down` - Move up/down
- `h/l` or `Left/Right` - Switch columns (Kanban)
- `Enter` - View details
- `Esc` - Back
- `q` - Quit

This provides a visual overview of work across all contexts.
